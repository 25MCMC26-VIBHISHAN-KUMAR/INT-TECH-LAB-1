Deep Learning Mini Project Announcement

Posted on 21 Jan 2026

Today, the Deep Learning faculty member, Srinivasa Rao Battula, announced that we have to work on a mini project. The mini project must be based on a research paper published after 2023. If a program already exists, we must enhance that project; if it is not implemented, then we must implement it ourselves.

Currently, this looks very challenging for me because I only know Python and some of its libraries. The project requires strong mathematics, especially topics like probability, vectors, calculus, statistics, machine learning, deep learning, and the use of deep learning libraries. At the same time, we also have to study other subjects, which makes it even more challenging since many of them are new to me.
